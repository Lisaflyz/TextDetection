Readme file for text detection program

1. Run compile.m to compile the c++ files.
2. Modify the image file path in demo.m and run it to see the visual result
3. Set the ICDAR2013 image file path in detect_icdar2013.m and run it to output
   the detection results to txt files
4. Run evaldet.m to compute the results of precision, recall and hmean 
   (Current it works only for Linux 64bits)
5. Run train_hog.m to train the SVM classifier.

If there are any problems, please contact us.
